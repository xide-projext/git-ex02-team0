# Git EX02

## 프로젝트 소개
이 프로젝트는 Git 사용법을 연습하기 위한 저장소입니다.

## 학습 목표
- Git 기본 명령어 익히기
- 브랜치 관리 방법 학습
- 협업 워크플로우 이해하기

## 주요 연습 내용
1. Git 기본 명령어
   - git init
   - git add
   - git commit
   - git push
   - git pull
   - git clone

2. 브랜치 관리
   - 브랜치 생성 및 삭제
   - 브랜치 병합
   - 충돌 해결

3. 협업 기능
   - Pull Request
   - Code Review
   - Issue 관리
